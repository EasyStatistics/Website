Welcome to EasyStats

EasyStats is a software that allows users to create R-based graphs without the need to know how to use R. It's main function is to identify optimal survival cut-points from multiple datasets. In addition, EasyStats creates Survival Plots, Correlations plots, Bar Graphs and Contigency tables. If you like R plots but you don't know how to generate them, this software is for you!

Software Features

Identify the optimal survival cut-point for multiple datasets

EasyStats allows users to upload multiple datasets to identify common cut-points. EasyStats creates a chart where you can see different percentiles as cut-points and the significance p-value for the Log-rank survival test. By doing this, users can identify optimal cut-points that work for multiple datasets. Instead of displaying the P-value, users also have the option to display the HR for the different percentiles.

Generate survival plots!

Once the users have identified the optimal cut-point, they can click on it and EasyStats automatically generates ready to publish Kaplan-Meier Survival Plots for all the datasets.

Compare association between markers!

EasyStats users can quickly generate marker correlation plots!

Create Bar plots

Aditionally, EasyStats can create Bar Plots to compare different markers between each other or the same marker within different categories.

Even Contingency Tables

Finally, if you are tired of counting how many patients are within one specific group, EasyStats can do it for you. Users can select the markers that the want to study and the categories of interest; then EasyStats generates a Contingency Table with the number of patients in each group splitted by the median value of the markers of interest.
